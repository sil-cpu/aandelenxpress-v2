API-koppeling: uw software direct verbonden

AX
AandelenXpress 16 april 2025 · 3 min lezen

Met de AandelenXpress REST API kunt u uw bestaande accountancysoftware rechtstreeks verbinden met ons platform. Zo hoeft u nooit meer handmatig klantegegevens over te typen en verloopt elk dossier gestroomlijnd vanuit uw eigen werkomgeving.

Wat biedt de AandelenXpress API?

De AandelenXpress API stelt u in staat om vanuit uw eigen software:

  • Nieuwe oprichtingsdossiers aan te maken en klantgegevens in te sturen;
  • De status van lopende dossiers realtime op te vragen;
  • Notificaties te ontvangen bij statuswijzigingen (webhooks);
  • Documenten (statuten, KvK-uittreksels) te downloaden zodra ze beschikbaar zijn;
  • Factuurgegevens op te vragen voor uw eigen boekhouding.

Beschikbare endpoints

De API is opgebouwd als een RESTful interface met JSON-responses. De voornaamste endpoints zijn:

  • POST /dossiers — Maak een nieuw dossier aan;
  • GET /dossiers/{id} — Haal de status en details op van een dossier;
  • GET /dossiers — Overzicht van alle dossiers van uw kantoor;
  • POST /webhooks — Registreer een webhook-URL voor statusmeldingen;
  • GET /documenten/{dossier_id} — Download beschikbare documenten.

Authenticatie: De API maakt gebruik van OAuth 2.0 Bearer Tokens. U genereert een API-sleutel in het partnerportaal onder Instellingen → API.

Integratiemogelijkheden

De API is geschikt om uw eigen processen en systemen te koppelen op basis van JSON en webhooks. Zo kunt u dossiergegevens, statusupdates en documenten geautomatiseerd verwerken binnen uw bestaande werkwijze.

Via de generieke REST API is maatwerkintegratie mogelijk. Onze technische documentatie (beschikbaar na inloggen in het partnerportaal) bevat alle benodigde informatie.

Technische vereisten en documentatie

De API is volledig gedocumenteerd via OpenAPI 3.0 (Swagger). Na activatie van uw partneraccount heeft u toegang tot de sandbox-omgeving, waarmee u de volledige API kunt testen zonder echte dossiers aan te maken. De volledige technische documentatie is beschikbaar op de developerportal in uw account.

Aan de slag in drie stappen

  1. Log in op het partnerportaal en navigeer naar Instellingen → API-toegang.
  2. Genereer een API-sleutel en sla deze veilig op.
  3. Raadpleeg de technische documentatie en verbind uw software met de sandbox-omgeving voor testen.

Heeft u hulp nodig bij de integratie? Ons technisch team staat voor u klaar via het contactformulier of de live chat in het portaal.
